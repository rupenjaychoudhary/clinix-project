About Clinix Theme

Clinix is a HTML theme, suited for hospitals and clinics. You can show your visitors environment of your health center. Clinix also allows you to list of the doctors working in your health center. Above all clinix comes with shop layout.

Pages

Clinix comes with pre-coded pages that you would need.

  • Home Page ( 2 layouts )
  • About Page
  • Contact Page
  • Team Page ( To list your staff )
  • Blog Page
  • Blog Single Page
  • Shop Page
  • Cart Page
  • Checkout Page
  • Product detail

CSS (cascade style sheet )

Theme is based on twitter bootstrap . and a few jquery plugins that comes with css files, so there are following css files used in the theme.

  • bootstrap
  • flexslider
  • animations
  • font-awesome
  • main

main.css is the files that has custome styles for clinix, in main.css code has been divided in to sections so that you could easily modify it. here is how you would see the code structure.

 

1-  General Styles
2-  Form Styling
3-  Spacing Helping Classes
4-  parallax
5-  Header
6-  Slider
7-  Inner Pages
8-  Shop
9-  Footer
10- Responsive

JS ( javascript )

It is recommended that you must get familiar with the API of the plugins used in this template that will help to modify things easily, Links to the plugin sites cold be found in the credit section.
